The GitHub How To Page

Below are the steps you need to know when working with GitHub.

GitHub is a site that allows users to house repos of their web pages.

Follow these steps to create your own GitHub account:

  • Go to www.github.com
  • Start by signing up. Provide an email address and make a username and password.
  • Once you have provided the necessary information, sign into your email address and verify your email. This will inform GitHub that you are in fact who you say you are.
  • Once you have been verified, sign into GitHub. Create another tab and install GitHub Desktop. Follow installation instructions.
  • After the installation pair your index file to your github. Take that file and clone it.
  • Once cloned save the file to your working folder. It should appear in Atom.
  • Begin to make changes to your html. Make sure that everything has been properly saved.
  • Go to GitHub Desktop and verify that your changes have been recognized. Green means you are good to go.
  • You can apply a description. And then click on Master. Sync.
  • Congratulations!! Your web page is now live on GitHub.

If you want to know more about all the great things that Github has to offer continue to explore the website, it's that easy. The future of web design starts here. No more headaches.
